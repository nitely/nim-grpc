# gRpc

Pure Nim [gRPC](https://grpc.io) implementation.
It works on top of [hyperx](https://github.com/nitely/nim-hyperx).

> [!NOTE]
> Beware this lib is in its infancy.

## Compatibility

> Nim +2.0
